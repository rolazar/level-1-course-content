School Bulletin

Each "person" gets a reference or "link" to the School Bulletin. It doesn't matter how they get it, but to model how it might pass from person to person, pass the reference to the bulletin just as you did in the telephone game.

This time, when the bulletin is changed, all the people will have up-to-date information. This is because an object is a reference type, and all the people have a reference to the same object. (In contrast to a value type.)

To work on this, go to 'bulletin.js', and change the values for the following variables:
